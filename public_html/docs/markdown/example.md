## 6. Lists

Unordered list:
- Item 1
- Item 2
    - Nested Item 2.1
    - Nested Item 2.2

* Item 3 (asterisk)

+ Item 4 (plus)

Ordered list:
1. First
2. Second
    1. Sub-item a
    2. Sub-item b
