Welcome to SBC Pi Tools

SBC Pi Tools are a Collection of Free and Open-Source Software (FOSS) and GNU (“GNU’s Not Unix”) tools for Single-Board Computers (SBCs). Use this site to find software, utilities, and guides to customize and configure a GNU/Linux OS on SBC hardware.

Quick definitions

  • FOSS — Free and Open-Source Software: software you are free to run, read, modify, and redistribute (examples of licenses: GPL, MIT, BSD, Apache).
  • GNU — GNUs Not Unix: a project and set of free userland tools (GCC, coreutils, glibc, bash, etc.) commonly used with the Linux kernel to form GNU/Linux systems.
  • SBC — Single-Board Computer: a complete computer built on a single circuit board (CPU, RAM, storage/I/O), e.g., Raspberry Pi, BeagleBone.
  • Armbian — a community project that provides a build system and prebuilt, optimized GNU/Linux images for single-board computers.
